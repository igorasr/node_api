# API
Api em NodeJS, com MySql

## Libs Utilizadas
* Express
* Driver Mysql2

## Como rodar o projeto

``` npm install```
``` npm start```

## Como subir o Container

```docker build -t api```

``` docker-compose up```

Subirão os serviços de API e Banco de Dados
